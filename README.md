# OTP SMS App

This is a simple OTP SMS app built with Next.js and Prisma.

## Getting Started

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
